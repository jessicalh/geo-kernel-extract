Command Line & REST Interface

Both nmr_extract (headless extraction) and nmr-viewer (Qt/VTK viewer) share the same JobSpec parser. All five modes work with both binaries.

Five Modes

1. Bare PDB

reduce protonates at the specified pH, ff14SB charges assigned.

nmr-viewer --pdb path/to/protein.pdb
nmr_extract --pdb path/to/protein.pdb --output /tmp/features

2. Pre-protonated PDB

From reduce, tleap, GROMACS, etc. Existing H atoms are kept.

nmr-viewer --protonated-pdb path/to/protein.pdb

3. ORCA DFT Run

Expects {root}.xyz, {root}.prmtop, optional {root}_nmr.out. Charges from prmtop. 

nmr-viewer --orca --root path/to/A0A7C5FAR6_WT

4. WT + ALA Mutant Pair

Each root expands independently. Computes mutation delta tensors.

nmr-viewer --mutant --wt path/to/WT --ala path/to/ALA

5. GROMACS Ensemble

TPR topology + pose directory. Full pipeline runs per frame.

nmr-viewer --fleet --tpr path/to/prod.tpr --poses path/to/poses

Common Flags

FlagDescription
--config path/to/params.toml Override calculator parameters (see CALCULATOR_PARAMETER_API.md)
--output DIR Write NPY feature arrays to directory (required for nmr_extract, optional for viewer)
--rest-port 0 Disable REST server (viewer only, default port 9147)

Launch Script

The script ui/launch_viewer.sh starts a UDP log listener in the background, sets DISPLAY=:1, and passes all arguments through to nmr-viewer. 

bash ui/launch_viewer.sh                              # 1ubq protonated (default)
bash ui/launch_viewer.sh --pdb myprotein.pdb          # bare PDB
bash ui/launch_viewer.sh --orca --root path/to/root   # ORCA DFT

Prerequisites

RuntimeEnvironment::Load() reads ~/.nmr_tools.toml at startup. This file must point to valid paths for mopac, reduce, OpenBabel, mkdssp, xtb, and the ff14SB parameter file.

REST Interface

The viewer listens on TCP port 9147 (tries 9147–9156) for newline-delimited JSON commands. Synchronous within the Qt event loop. Starts automatically; disable with --rest-port 0. 

# Quick test
echo '{"cmd":"status"}' | nc -q1 localhost 9147

Response Format

{"ok": true,  "result": { ... }}
{"ok": false, "error": "description"}

Session Commands

CommandDescription
status Protein name, atom/ring/residue counts, compute state, overlay visibility
load_pdb Build JobSpec with mode=Pdb, trigger compute. Params: path
export_features Write all NPY arrays to directory. Fleet creates frame_N/ subdirs. Params: path

Camera Commands

CommandParametersDescription
get_camera — Returns position, focal_point, view_up, distance, view_angle
set_camera position, focal_point, view_up (all optional) Update camera; omitted fields keep current value
orbit azimuth, elevation (degrees) Rotate camera around focal point
reset_view — Reset to default view
look_at_ring ring (index), view (side/top/edge), distance Frame camera on aromatic ring
look_at_atom atom (index), distance Re-centre camera on atom

Rendering Commands

CommandParametersDescription
set_render_mode mode: ball_stick, stick (alias liquorice) Molecular representation
set_glyph_scale scale (float) Atom/bond glyph size
set_opacity value (0–1) Global opacity

Overlay Commands

CommandParametersDescription
show_ringsvisibleAromatic ring overlay
show_bondsvisiblePeptide bond overlay
show_butterflyvisibleBiot-Savart magnetic field visualisation
show_field_grid shielded, deshielded (or visible for both) Isosurface regions
set_iso_thresholdvalue (0–1)Isosurface contour level

Data & Output Commands

CommandParametersDescription
list_rings — Array of ring objects: index, type, residue, center, normal, radius, intensity
get_log lines (or first/last) Operations log panel contents
screenshot path Save current render as PNG. Returns path, width, height

Scripting Example

# Load a protein, wait for compute, take a screenshot
echo '{"cmd":"load_pdb","path":"/data/1ubq.pdb"}' | nc -q1 localhost 9147
sleep 10
echo '{"cmd":"status"}' | nc -q1 localhost 9147
echo '{"cmd":"show_butterfly","visible":true}' | nc -q1 localhost 9147
echo '{"cmd":"look_at_ring","ring":0,"view":"side","distance":12}' | nc -q1 localhost 9147
echo '{"cmd":"screenshot","path":"/tmp/butterfly.png"}' | nc -q1 localhost 9147
echo '{"cmd":"export_features","path":"/tmp/features"}' | nc -q1 localhost 9147
