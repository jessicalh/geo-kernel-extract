3. Calculator Pipeline

OperationRunner::Run sequences all results in dependency order. Tier 0 provides foundation data (geometry, spatial index, charges, DSSP, APBS, MOPAC). Tier 1 runs the 8 classical geometric kernel calculators plus 2 MOPAC-derived calculators, each producing full rank-2 tensor output.
