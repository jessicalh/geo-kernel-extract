Loading

load(path)

Load an extraction directory into a fully typed Protein.

Validates every file against the catalog. Warns on unregistered files (forward-compatible). Errors on missing required files.

Return type:

Protein

Parameters:

path (str | Path)

class Protein(protein_id, n_atoms, pos, element, residue_type, residue_index, biot_savart, haigh_mallion, pi_quadrupole, dispersion, ring_susceptibility, ring_contributions=None, ring_geometry=None, mcconnell=None, coulomb=None, hbond=None, dssp=None, mopac=None, apbs=None, orca=None, delta=None)

All extracted features for one protein conformation.

Every field is typed. Optional groups are None when the extractor did not produce them (MOPAC, APBS, Orca, Delta).
