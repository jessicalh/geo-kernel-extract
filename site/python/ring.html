Per-Ring Data

Sparse (P, 57) table of per-(atom, ring) pair contributions, plus a (R, 10) ring geometry reference table.

RingContributions

class RingContributions(data)

Sparse per-(atom, ring) pair contributions.

Shape (P, 57) where P = number of evaluated (atom, ring) pairs. Each row carries geometry, all ring current kernels as SphericalTensor views, and dispersion scalars.

Use for_atom() and for_ring_type() to filter rows.

Parameters:

data (ndarray) – numpy array of shape (P, 57).

RingGeometry

class RingGeometry(data)

Per-ring geometry reference table.

Shape (R, 10) where R = number of rings in the protein. One row per ring: index, type, residue, center, normal, radius.
