The format contract between C++ and Python. One entry per NPY file the extractor can produce.

class ArraySpec(stem, group, wrapper, cols, required, description)

Metadata for one NPY file produced by the C++ extractor.

Parameters:

  • stem (str)

  • group (str)

  • wrapper (type)

  • cols (int | None)

  • required (bool)

  • description (str)

CATALOG is a dict[str, ArraySpec] mapping every NPY filename stem to its metadata. 53 entries covering identity, ring calculators, bond calculators, MOPAC, APBS, Orca DFT, and mutation delta arrays.

from nmr_extract import CATALOG
for stem, spec in CATALOG.items():
    print(f"{stem:30s} {spec.group:20s} required={spec.required}")
