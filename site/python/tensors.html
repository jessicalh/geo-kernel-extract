SphericalTensor

class SphericalTensor(data)

Irreducible spherical tensor: T0 (L=0) + T1 (L=1) + T2 (L=2).

Shape (*, 9) packed as [T0, T1[3], T2[5]].

T2 component ordering: m = -2, -1, 0, +1, +2 (matches e3nn). Normalization: isometric (Frobenius-preserving).

Parameters:

data (ndarray) – numpy array with last dimension 9.

Raw numpy array (*, 9).

Irreps("1x0e+1x1o+1x2e").
