2//
3// MopacMcConnellResult: Bond anisotropy shielding weighted by MOPAC bond order.
4//
5// Same McConnell kernel as McConnellResult:
6// M_ab = 9 cos_theta d_hat_a b_hat_b
7// - 3 b_hat_a b_hat_b
8// - (3 d_hat_a d_hat_b - delta_ab)
9//
10// Each bond's contribution is weighted by its MOPAC Wiberg bond order:
11// total += bond_order_k * M_ab_k / r_k^3
12//
13// The bond order is a measured QM quantity (electron density sharing),
14// not a parameter. It modulates which bonds dominate the angular sum.
15// A C=O with order 1.8 contributes more kernel than one with 1.2.
16// The model learns Delta_chi per category for this weighted kernel.
17//
18// Same per-category decomposition as McConnellResult:
19// PeptideCO, PeptideCN, BackboneOther, SidechainCO, Aromatic.
20//
21// Dependencies: MopacResult (bond orders), SpatialIndexResult, GeometryResult.
