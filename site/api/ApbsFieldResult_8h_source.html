1#pragma once
2//
3// ApbsFieldResult: electrostatic E-field and EFG at each atom.
4//
5// Dependencies: ChargeAssignmentResult (charges and radii).
6//
7// Stores per atom: apbs_efield (Vec3), apbs_efg (Mat3),
8// apbs_efg_spherical (SphericalTensor).
9//
10// Linearised Poisson-Boltzmann solve via the APBS C bridge (apbs_bridge.h).
11// The bridge takes in-memory arrays of positions, charges, and radii and
12// returns a potential grid in kT/e. E-field and EFG are extracted by
13// central-difference interpolation on the grid.
14//
15// No fallback. If APBS fails, Compute() returns nullptr. Solvated fields
16// require a working PB solver — substituting vacuum Coulomb would silently
17// produce a different physical quantity in different units.
18//
19// E-field from grid: E = -grad(phi) via central differences
20// EFG from grid: dE_a/dx_b via central differences of E-field
21// Traceless projection applied (removes self-potential artifact).
22//
23// Units stored on ConformationAtom: V/A for E-field, V/A^2 for EFG.
24// Converted from APBS native kT/(e*A) by multiplying by kT/e = 0.025693 V
25// at 298.15 K. This makes APBS fields directly comparable to CoulombResult.
26//
