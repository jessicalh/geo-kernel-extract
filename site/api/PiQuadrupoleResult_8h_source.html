2//
3// PiQuadrupoleResult: quadrupole EFG kernel from aromatic ring pi-electrons.
4//
5// For each aromatic ring within range of each atom, computes the EFG
6// tensor from a point axial quadrupole at the ring center along the
7// ring normal. The pi-electron cloud creates a quadrupole electric
8// field; its EFG at nearby nuclei couples to shielding via the
9// Buckingham gamma coefficient.
10//
11// Derivation: the potential from an axial quadrupole Theta along n is
12// Phi = Theta * (3 cos^2 theta - 1) / (2 r^3)
13//
14// The EFG V_ab = -d^2 Phi / dx_a dx_b is obtained via Stone's T-tensor:
15// V_ab = -(1/3) Theta_cd T_abcd
16// where T_abcd = d^4(1/r) / dx_a dx_b dx_c dx_d is the rank-4 interaction
17// tensor (Stone, Theory of Intermolecular Forces, OUP 2013, Ch. 3).
18//
19// For Theta_cd = Theta(3 n_c n_d - delta_cd)/2, the delta contraction
20// vanishes by Laplace (T_abcc = nabla^2 nabla^2 (1/r) = 0), giving
21// the geometric kernel:
22//
23// G_ab = 105 dn^2 d_a d_b / r^9
24// - 30 dn (n_a d_b + n_b d_a) / r^7
25// - 15 d_a d_b / r^7
26// + 6 n_a n_b / r^5
27// + delta_ab (3/r^5 - 15 dn^2/r^7)
28//
29// where d = r_atom - r_center, r = |d|, dn = d . n.
30//
31// Properties:
32// - Symmetric: G_ab = G_ba
33// - Traceless: Tr(G) = 0 (Laplace equation outside sources)
34// - Pure T2: T0 = 0, T1 = 0
35// - Leading radial decay: 1/r^5
36//
37// Also stores the scalar (3 cos^2 theta - 1) / r^4 for the Buckingham
38// A-term (E-field from quadrupole -> T0 isotropic shift). This is a
39// SEPARATE physical effect from the EFG -> T2 coupling.
40//
41// Units: G in Angstrom^-5 (d in A, r in A). Scalar in Angstrom^-4.
42//
