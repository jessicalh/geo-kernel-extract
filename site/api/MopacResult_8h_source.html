3// MopacResult: PM7+MOZYME semiempirical calculation via libmopac.
4//
5// Provides per-atom Mulliken charges, orbital populations (s, p),
6// Wiberg bond orders (continuous, conformation-dependent), heat of
7// formation, and dipole moment.
8//
9// Dependencies: none.
10//
11// Calls MOPAC via run_mopac_from_input() (linked libmopac, in-process).
12// Writes a .mop input file to RuntimeEnvironment::TempFilePath,
13// MOPAC writes the .out alongside it, we parse both structured API
14// output and text output for orbital populations.
15//
16// Replaces XtbChargeResult. Same tier in OperationRunner (0.5).
17//
18// Future consumers:
19// MopacCoulombResult — ChargeAt(i) for QM-derived EFG
20// MopacBondAnisotropyResult — BondOrder(a,b) for delta-chi modulation
21// DipolarNearFieldFilter — mopac_p_pop for lobe offset calibration
