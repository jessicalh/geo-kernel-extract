3// RuntimeEnvironment: the running program's configuration.

5// PRECONDITION: Call RuntimeEnvironment::Load() once at program startup.
6// Every accessor checks this and aborts if you forgot. There is
7// no silent fallback. If the TOML doesn't exist, Load() still
8// runs — it reads env vars and PATH, logs what it found, and
9// warns about what's missing. But it must be called.
10//
11// TOML location: ~/.nmr_tools.toml (or pass explicit path).
12// Every resolved value is logged at Load() time so you can see
13// exactly what the program thinks its environment is.
14//
15// Live surface (2026-04-05):
16// ff14sb_params — BuildFromPdb charge assignment
17// tmpdir — temp files for MOPAC work directories
18//
19// MOPAC is linked (libmopac.so), not a binary path. No accessor needed.
