2//
3// DispersionResult: London dispersion (van der Waals) tensors from ring vertices.
4//
5// For each aromatic ring within range of each atom, computes the
6// anisotropic dispersion kernel by summing over ring vertex atoms:
7//
8// K_ab = sum_vertices S(r) * (3 d_a d_b / r^8 - delta_ab / r^6)
9// scalar = sum_vertices S(r) / r^6
10//
11// where d = r_atom - r_vertex, r = |d|, S(r) is a smooth switching
12// function (CHARMM form, Brooks et al. 1983) that tapers to zero at
13// DISP_VERTEX_R_CUT = 5.0 A.
14//
15// Properties of the tensor kernel per vertex:
16// - Symmetric: K_ab = K_ba
17// - Traceless: Tr(K) = S(r) * (3r^2/r^8 - 3/r^6) = 0
18// - Pure T2 per vertex; sum over vertices is also traceless
19//
20// The isotropic 1/r^6 scalar (attractive VdW) is stored separately.
21// The tensor captures the anisotropic part — the angular dependence
22// of dispersion from the discrete vertex positions around the ring.
23//
24// Filters:
25// - DipolarNearFieldFilter at ring level (source_extent = ring diameter):
26// same physics as all other ring calculators.
27// - Through-bond vertex exclusion: atoms bonded to any ring vertex are
28// excluded because the 1/r^6 kernel models through-space interaction,
29// not through-bond coupling. Uses protein bond connectivity, not
30// a distance heuristic.
31// - Smooth switching function: C^1 taper from 4.3A to 5.0A per vertex.
32// No hard cutoff discontinuity. Essential for MD frame continuity.
33//
34// Unit C6 = 1 — parameter is learnable per ring type.
35// Units: tensor in Angstrom^-6, scalar in Angstrom^-6.
36//
