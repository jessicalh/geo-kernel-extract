2//
3// BiotSavartResult: Johnson-Bovey ring current B-field and shielding tensors.
4//
5// For each aromatic ring within 15A of each atom, computes:
6//
7// 1. B-field from Johnson-Bovey double-loop model (wire segments in SI,
8// two current loops at +/- lobe_offset from ring plane, each I/2).
9//
10// 2. Geometric kernel G_ab = n_b * B_a * PPM_FACTOR (rank-1 outer product
11// of ring normal with B-field). Dimensionless.
12//
13// 3. SphericalTensor decomposition of G:
14// T0 = (n . B) * PPM_FACTOR / 3 (isotropic ring current shift)
15// T1 != 0 (asymmetric: n_b B_a != n_a B_b)
16// T2 != 0 (angular anisotropy)
17//
18// Boyd & Skrynnikov JACS 2002 124:1832 showed that both T0 (eq 1, known
19// since Johnson-Bovey 1957) and the off-diagonal components (eq 2, new)
20// are needed for the full shielding tensor. Our G_ab = n_b B_a construction
21// produces exactly their eq 3 decomposition.
22//
23// Sign convention: G_ab = -n_b * B_a * PPM_FACTOR. The minus sign comes
24// from the shielding tensor definition sigma_ab = -dB_a^sec/dB_{0,b}.
25// With this convention, sigma = I * G gives the correct physical sign
26// using literature ring current intensities (I < 0 for diamagnetic).
27// Verified: I=-12, atom 3A above PHE -> sigma = +1.40 ppm (shielded).
28// In the ring plane at 5A -> sigma = -0.16 ppm (deshielded).
29//
30// Units: positions in Angstroms, current in nanoamperes, B computed in
31// Tesla (SI internally), G dimensionless (after PPM_FACTOR).
32//
