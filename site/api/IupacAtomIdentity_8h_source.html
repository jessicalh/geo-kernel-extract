3// IupacAtomIdentity: ground truth validator for IUPAC protein atom naming.
4//
5// This module answers: "What IS atom CA in alanine?" Not by convention,
6// not by parsing -- by the IUPAC definition of the amino acid.
7//
8// For each atom in each amino acid, this provides:
9// - Element (redundant with name, but explicit)
10// - Whether it's backbone or sidechain
11// - Whether it's a ring member
12//
13// This is the VALIDATOR. When we load a PDB and assign atom names, this
14// module checks: "You say this atom is CA. Is it element C? Is it backbone?"
15// If not, something is wrong.
16//
17// Usage:
18// const auto* id = IupacAtomIdentity::Lookup("ALA", "CA");
19// id->element; // Element::C
20// id->is_backbone; // true
21// id->is_ring_member;// false
22//
23// std::string err = IupacAtomIdentity::Validate("ALA", "CA", Element::C, true);
