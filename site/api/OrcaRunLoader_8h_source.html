2//
3// OrcaRunLoader: load a protein from an ORCA DFT run.
4//
5// An ORCA run always has:
6// - PDB: AlphaFold/crystal structure (heavy atoms, sequence identity)
7// - XYZ: protonated coordinates from tleap (all atoms, ORCA geometry)
8//
9// It may also have:
10// - prmtop: AMBER topology (authoritative atom names, residues, charges)
11// - NMR output: per-atom shielding tensors (loaded separately)
12// - tleap artifacts: script, log, amber PDB, inpcrd
13//
14// Two loading paths:
15// WITH prmtop (all 723 proteins): prmtop provides the protonated atom list
16// (names, residues, element). XYZ provides positions.
17// As of 2026-04-02, prmtop regenerated via tLeap ff14SB for the 253
18// proteins that were previously missing them. All proteins now use
19// this path.
20// WITHOUT prmtop (legacy fallback): PDB provides sequence. AminoAcidType
21// provides the canonical atom list per residue (including H). XYZ
22// provides positions. Atoms matched by residue structure and element.
23//
24// Both paths produce: a Protein with the full protonated atom list and
25// one conformation with XYZ positions. Charges and NMR tensors attach
26// separately as ConformationResults from whatever source is available.
27//
