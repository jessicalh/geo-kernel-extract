3// ProtonationDetectionResult: detect protonation state of titratable
4// residues by examining which hydrogen atoms are present in the structure.
5//
6// PDB LOADING BOUNDARY code. Reads atom name strings ONCE to determine
7// which hydrogens are present on titratable sidechains, then stores
8// the result as a typed protonation_variant_index on each Residue.
9//
10// Detection logic (from hydrogen names in the PDB coordinate set):
11// HIS: "HD1" present -> HID, "HE2" present -> HIE, both -> HIP, neither -> ambiguous
12// ASP: "HD2" present -> ASH (protonated), absent -> ASP (charged)
13// GLU: "HE2" present -> GLH, absent -> GLU (charged)
14// CYS: SG bonded to another SG -> CYX, "HG" present -> CYS
15// LYS: all three HZ -> LYS (charged), "HZ3" absent -> LYN
16//
17// Dependencies: none (reads topology/atom names only).
18//
19// After this result, DetectAromaticRings should read from
20// protonation_variant_index for HIS tautomer, not do its own string check.
21//
