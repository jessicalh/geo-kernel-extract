2//
3// MopacCoulombResult: Coulomb E-field and EFG from MOPAC QM charges.
4//
5// Same dipolar kernel as CoulombResult:
6// E_a(i) = ke * sum_{j!=i} q_j * (r_i - r_j)_a / |r_i - r_j|^3
7// V_ab(i) = ke * sum_{j!=i} q_j * [3 (r_i-r_j)_a (r_i-r_j)_b / |r_i-r_j|^5
8// - delta_ab / |r_i-r_j|^3]
9//
10// Different charge source: reads mopac_charge (PM7 Mulliken, conformation-
11// dependent) instead of partial_charge (ff14SB, fixed per atom type).
12// The T2 angular pattern differs because MOPAC charges respond to the
13// local electronic environment. The model learns gamma_mopac.
14//
15// Decomposed by source: backbone, sidechain, aromatic (same as CoulombResult).
16// No APBS solvent subtraction (APBS was solved with ff14SB charges).
17//
18// Units: V/A (E-field), V/A^2 (EFG), same as CoulombResult.
19//
20// Dependencies: MopacResult (charges on ConformationAtom),
21// SpatialIndexResult (ensures geometry is resolved).
22//
