2//
3// RingSusceptibilityResult: ring magnetic susceptibility anisotropy tensors.
4//
5// For each aromatic ring within range of each atom, computes the full
6// shielding tensor from the ring's bulk diamagnetic susceptibility
7// anisotropy, treated as a point magnetic dipole at the ring center.
8//
9// Same derivation as McConnell (GEOMETRIC_KERNEL_CATALOGUE.md) with
10// b_hat replaced by ring normal n_hat:
11//
12// M_ab = 9 cosθ d̂_a n_b - 3 n_a n_b - (3 d̂_a d̂_b - δ_ab)
13//
14// Shielding contribution = M_ab / r³ (Angstrom⁻³)
15//
16// This tensor is ASYMMETRIC (T1 ≠ 0) and NON-TRACELESS (T0 = scalar f).
17// Same structure as McConnell, different source geometry.
18//
19// The symmetric traceless dipolar kernel K_ab is also stored separately
20// for features.
21//
