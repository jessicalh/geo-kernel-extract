2//
3// CovalentTopology: the geometry-derived connectivity of a protein.
4//
5// This is where geometric facts (distances between atoms) become
6// symbolic facts (bonds, bond categories, H parents). This happens
7// once, from one set of positions, and all conformations share it.
8//
9// The assumption is that covalent topology is invariant across
10// conformations — true for classical MD, constant-pH excepted.
11//
12// Resolve() takes the output of symbolic topology resolution
13// (atoms with canonical names, residues with backbone indices,
14// rings with typed membership) plus one set of positions.
15// It returns a self-contained object.
16//
17// Protein delegates BondAt/BondCount through this object and
18// copies per-atom bond_indices and parent_atom_index back to
19// Atom objects for convenient calculator access.
20//
21
