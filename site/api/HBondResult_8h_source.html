2//
3// HBondResult: hydrogen bond dipolar shielding tensors.
4//
5// For each atom involved in an H-bond (identified by DSSP), computes
6// the full shielding tensor from the dipolar interaction with the
7// H-bond partner. Same derivation as McConnell/RingSusceptibility
8// (GEOMETRIC_KERNEL_CATALOGUE.md) with b_hat replaced by the
9// donor→acceptor direction h_hat.
10//
11// H-bond identification comes from DsspResult, which provides
12// backbone H-bond partners via the Kabsch-Sander energy criterion.
13// The H-bond direction h_hat is computed from the actual atomic
14// positions (donor N → acceptor O for backbone H-bonds).
15//
16// The full tensor:
17// M_ab = 9 cosθ d̂_a h_b - 3 h_a h_b - (3 d̂_a d̂_b - δ_ab)
18// Shielding contribution = M_ab / r³ (Angstrom⁻³)
19//
20// where d = r_atom - r_partner, h_hat = direction from donor to acceptor.
21//
22// Asymmetric, non-traceless, T0+T1+T2 all non-zero.
23//
