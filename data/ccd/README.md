# data/ccd/

PDB Chemical Component Dictionary. Reference data, freely re-fetchable
from wwPDB; not redistributed in this repo.

## Fetch

    cd data/ccd
    curl -O https://files.wwpdb.org/pub/pdb/data/monomers/components.cif.gz
    gunzip -k components.cif.gz   # -k keeps the .gz alongside the .cif

Sizes: `components.cif.gz` ~89 MB, `components.cif` ~376 MB.
